Outils : Docker, Atlas et mongosh

Un environnement reproductible pour pratiquer MongoDB sans dépendre de l'installation locale de chaque machine.

Les outils du cours

OutilRôle
Docker ComposeLance MongoDB et Mongo Express.
MongoDBBase NoSQL orientée documents.
mongoshShell officiel pour écrire les commandes.
Mongo ExpressInterface web légère de vérification.
AtlasVersion cloud managée de MongoDB.

Démarrer l'environnement

docker compose up -d
docker compose ps

MongoDB écoute sur `localhost:27017`. Mongo Express est disponible sur `http://localhost:8083`.

Se connecter avec mongosh

docker compose exec mongodb mongosh \
  "mongodb://root:rootpass@localhost:27017/nyc_food?authSource=admin"
db
show dbs
show collections
exit

Local ou Atlas ?

Docker local

  • Exercices en classe
  • Reproductibilité
  • Réinitialisation facile
  • Données identiques pour tous

Atlas

  • Cluster managé
  • Utilisateurs et réseau
  • Connexion `mongodb+srv`
  • Monitoring et sauvegardes
