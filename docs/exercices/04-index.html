Liste 04

Index et performance

Mesurer avec `explain()`, repérer un `COLLSCAN` et créer des index adaptés.

Mesurer une requête

const restaurantId = db.restaurants.findOne({}, { restaurant_id: 1 }).restaurant_id

db.orders.find({
  restaurant_id: restaurantId,
  status: "paid"
}).sort({ created_at: -1 }).explain("executionStats")
  • `executionTimeMillis`
  • `totalDocsExamined`
  • `totalKeysExamined`
  • `stage`

Créer des index

db.orders.createIndex({ restaurant_id: 1, status: 1, created_at: -1 })
db.restaurants.createIndex({ "ratings.overall": -1, price_for_two: 1 })
db.restaurants.createIndex({ tags: 1, "ratings.food": -1 })
db.review_details.createIndex({ sentiment: 1, rating: -1 })
db.neighborhoods.createIndex({ center: "2dsphere" })

Index géospatial

`neighborhoods.center` contient un point GeoJSON. L'index `2dsphere` permet des requêtes de proximité avec `$near`.

db.neighborhoods.find({
  center: {
    $near: {
      $geometry: { type: "Point", coordinates: [-73.9855, 40.7580] },
      $maxDistance: 5000
    }
  }
}).explain("executionStats")
