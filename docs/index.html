MongoDB Introduction

Un parcours de contenu pour découvrir MongoDB, manipuler `mongosh`, interroger des documents avec `find`, construire des agrégations et pratiquer sur des listes d'exercices.

Parcours

Le support suit un fil rouge autour d'une base `nyc_food`. Les cours introduisent les concepts, puis les exercices permettent de les appliquer directement dans le sandbox Docker.

Cours

Six chapitres structurés : outils, jeu de données, schéma, mongosh, find et aggregate.
