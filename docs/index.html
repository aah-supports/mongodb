MongoDB Introduction

Un parcours de contenu pour découvrir MongoDB, comprendre ses cas d'usage, manipuler `mongosh`, interroger des documents avec `find`, construire des agrégations et pratiquer sur des listes d'exercices.

Parcours

Le support suit un fil rouge autour d'une base `nyc_food`. Les cours introduisent les concepts, puis les exercices permettent de les appliquer directement dans le sandbox Docker.

Cours

Huit chapitres structurés : introduction, outils, jeu de données, schéma, mongosh, find, aggregate et droits.

Exercices

Listes d'exercices apprenants. La correction `find` est accessible depuis sa liste.

Collections du sandbox

  • `nyc_restaurant_reviews_raw` : données brutes OpenIntro/Zagat.
  • `restaurants` : vue documentaire orientée établissement.
  • `reviews` : avis et notes agrégées.
  • `neighborhoods` : collection pédagogique annexe.
  • `orders` : commandes simulées pour volume et performances.
  • `review_details` : avis clients détaillés simulés.
  • `events` : événements applicatifs simulés.
