Cours 05

Analyser avec aggregate

db.collection.aggregate([
  { stage1 },
  { stage2 },
  { stage3 }
])

Chaque étape reçoit des documents, les transforme, puis transmet le résultat.

Stages essentiels

`$match`

Filtre les documents.

`$project`

Choisit et renomme les champs.

`$set`

Ajoute un champ calculé.

`$group`

Regroupe et calcule des agrégats.

`$unwind`

Éclate un tableau.

`$lookup`

Relie deux collections.

Regrouper

db.restaurants.aggregate([
  {
    $group: {
      _id: "$cuisine",
      average_food: { $avg: "$ratings.food" },
      average_overall: { $avg: "$ratings.overall" },
      restaurants: { $sum: 1 }
    }
  },
  { $sort: { average_overall: -1 } }
])

Relier avec `$lookup`

db.orders.aggregate([
  { $match: { status: "paid" } },
  { $group: { _id: "$restaurant_id", revenue: { $sum: "$amount" }, orders: { $sum: 1 } } },
  {
    $lookup: {
      from: "restaurants",
      localField: "_id",
      foreignField: "restaurant_id",
      as: "restaurant"
    }
  },
  { $unwind: "$restaurant" }
])

Matérialiser avec `$merge`

db.orders.aggregate([
  { $match: { status: "paid" } },
  { $group: { _id: "$restaurant_id", revenue: { $sum: "$amount" } } },
  {
    $merge: {
      into: "restaurant_kpis",
      whenMatched: "replace",
      whenNotMatched: "insert"
    }
  }
])

`aggregate` transforme des documents en indicateurs métier : revenus, satisfaction, classements et KPI.
